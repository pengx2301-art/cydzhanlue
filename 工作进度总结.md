**认证相关：**
- POST /api/auth/login - 登录
- POST /api/auth/logout - 登出
- GET /api/auth/me - 获取当前用户信息

**订单相关：**
- GET /api/orders - 获取订单列表
- POST /api/orders - 创建订单
- GET /api/orders/:id - 获取订单详情
- PUT /api/orders/:id/status - 更新订单状态
- GET /api/orders/:id/balance - 查询订单用户余额（需重启）
- POST /api/orders/:id/recharge - 发起接口充值

**用户相关：**
- GET /api/members - 获取用户列表
- POST /api/members - 添加用户
- PUT /api/members/:id - 更新用户信息
- POST /api/members/:id/balance - 调整用户余额

**产品相关：**
- GET /api/products - 获取产品列表
- GET /api/categories - 获取产品分类
- GET /api/suppliers - 获取供应商列表

**日志相关：**
- GET /api/balance-logs - 获取余额变动日志
- GET /api/operation-logs - 获取操作日志
