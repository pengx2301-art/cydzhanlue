## 🔧 技术实现

### 1. 客客帮签名算法
```javascript
generateSignature(params) {
    // 排除sign参数
    const signParams = {};
    Object.keys(params).forEach(key => {
        if (key !== 'sign') {
            signParams[key] = params[key];
        }
    });

    // 按参数名首字母升序排序
    const sortedKeys = Object.keys(signParams).sort();
    
    // 拼接字符串
    let signString = '';
    sortedKeys.forEach(key => {
        signString += key + signParams[key];
    });

    // 追加AppSecret
    signString += this.appSecret;

    // MD5加密
    return crypto.createHash('md5').update(signString).digest('hex');
}
```

## 🚀 使用示例

### 1. 创建客客帮供货商实例
```javascript
const KekebangSupplier = require('./kekebang-supplier');

const supplier = new KekebangSupplier({
    apiUrl: 'https://purchase.kekebang.com.cn',
    supplyApiUrl: 'https://openapi.kekebang.com.cn',
    appKey: 'your_app_key',
    appSecret: 'your_app_secret'
});
```

### 2. 获取商品列表
```javascript
const products = await supplier.getProductList({
    bId: 1, // 话费
    page: 1,
    pageSize: 100
});

console.log(products.data);
```

### 3. 提交话费订单
```javascript
const order = await supplier.submitOrder({
    bizCode: 1, // 话费
    externalOrderNo: 'ORDER20250119001',
    skuCode: 'SKU0001',
    callbackUrl: 'http://your-site.com/callback',
    data: {
        account: '15555555555',
        mno: '移动',
        province: '北京'
    }
});
```

### 4. 查询订单状态
```javascript
const orderDetail = await supplier.getOrderDetail('ORDER20250119001', 1);

console.log('订单状态:', orderDetail.data.order_state);
console.log('充值结果:', orderDetail.data.return_msg);
```
