## 📋 API接口清单

### 用户后台API
```
POST   /api/user/login                    # 用户登录
GET    /api/user/dashboard                 # 仪表盘数据
GET    /api/user/products                  # 获取商品列表
POST   /api/user/order/single             # 单个下单
POST   /api/user/order/bulk               # 批量下单
GET    /api/user/orders                    # 订单列表
GET    /api/user/order/:id                 # 订单详情
POST   /api/user/topup/apply               # 申请加款
GET    /api/user/topup/applications        # 加款记录
```

### 总后台API
```
POST   /api/admin/login                    # 管理员登录
GET    /api/admin/dashboard                 # 仪表盘数据
POST   /api/admin/topup/approve            # 审核加款
GET    /api/admin/topup/applications       # 加款申请列表
```

### 自来单API
```
POST   /api/zilaidan/configure             # 配置平台
GET    /api/zilaidan/orders                # 订单列表
POST   /api/zilaidan/order/process         # 处理订单
POST   /api/zilaidan/order/batch           # 批量处理
GET    /api/zilaidan/statistics            # 订单统计
POST   /api/zilaidan/mifenghuiyun/orders   # 拉取蜜蜂汇云订单
POST   /api/zilaidan/kekebang/orders      # 拉取客客帮订单
```

### 供货商API
```
POST   /api/supplier/add                    # 添加供货商
PUT    /api/supplier/:id                   # 更新供货商
DELETE /api/supplier/:id                   # 删除供货商
PUT    /api/supplier/:id/toggle            # 启用/禁用
POST   /api/supplier/:id/test              # 测试接口
GET    /api/supplier/list                  # 供货商列表
GET    /api/supplier/:id/stats             # 供货商统计
GET    /api/supplier/search                # 搜索供货商
```

### 系统API

## 🎯 核心功能演示

### 用户后台流程
1. **登录** → 获取认证Token
2. **仪表盘** → 查看余额、订单数、消费金额
3. **浏览商品** → 查看可用商品列表
4. **下单** → 选择商品，填写账号，提交订单
5. **查看订单** → 查看订单状态和详情
6. **加款申请** → 申请充值，等待审核

### 自来单流程
1. **配置平台** → 配置蜜蜂汇云、客客帮等
2. **拉取订单** → 从平台拉取待处理订单
3. **处理订单** → 执行充值操作
4. **上报状态** → 将订单状态回调给平台
5. **批量处理** → 批量处理多个订单

### 供货商管理流程
1. **添加供货商** → 配置供货商API信息
2. **测试接口** → 验证API连接
3. **管理商品** → 从供货商拉取商品
4. **统计分析** → 查看供货商业绩
