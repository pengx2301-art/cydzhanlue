### 步骤2: 配置SSH连接 ⏳
需要手动配置SSH密钥或使用密码登录。

**配置SSH密钥（推荐）**:
```powershell

ssh-keygen -t ed25519 -C "workbuddy@claw"

# 复制公钥到服务器
type $env:USERPROFILE\.ssh\id_ed25519.pub | ssh root@81.70.208.147 "cat >> ~/.ssh/authorized_keys"
```
