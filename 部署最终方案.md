## 🚀 一键部署（最新版）

### 方法1：使用curl（推荐）

```bash
cd ~ && curl -O https://raw.githubusercontent.com/pengx2301-art/cydzhanlue/main/deploy-remote.sh && chmod +x deploy-remote.sh && ./deploy-remote.sh
```

### 方法2：使用wget

```bash
cd ~ && wget https://raw.githubusercontent.com/pengx2301-art/cydzhanlue/main/deploy-remote.sh && chmod +x deploy-remote.sh && ./deploy-remote.sh
```

### 方法3：手动执行（如果下载失败）

在服务器上直接执行以下命令：

```bash
cd /app

# 1. 备份现有文件
mkdir -p backup_$(date +%Y%m%d_%H%M%S)
for file in api-server.js product-connection.js order-module.js package.json; do
    [ -f "$file" ] && cp "$file" "backup_$(date +%Y%m%d_%H%M%S)/"
done

# 2. 安装依赖
sudo apt update
sudo apt install -y git curl

# 3. 安装Node.js（如果未安装）
if ! command -v node &> /dev/null; then
    curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
    sudo apt install -y nodejs
fi

# 4. 安装PM2（如果未安装）
if ! command -v pm2 &> /dev/null; then
    sudo npm install -g pm2
fi

# 5. 克隆代码（使用HTTPS，无需SSH密钥）
git clone https://github.com/pengx2301-art/cydzhanlue.git temp

# 6. 复制文件
cp temp/api-server.js .
cp temp/product-connection.js .
cp temp/order-module.js .
cp temp/package.json .
cp temp/product-connection.html .
cp temp/order-page.html .

# 7. 清理临时目录
rm -rf temp

# 8. 安装依赖
npm install

# 9. 重启服务
pm2 restart cyd-admin || pm2 start api-server.js --name cyd-admin
pm2 save

# 10. 查看状态
pm2 status
pm2 logs cyd-admin --lines 20
```
