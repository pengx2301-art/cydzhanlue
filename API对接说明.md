# 充易达系统 v3.0.0 - API对接说明

## 📋 业务角色说明

### 1. 自来单系统
**我们作为供货商**

- **蜜蜂汇云**：采购商
  - 我们对接蜜蜂汇云的**卖家API**（供应API）
  - 蜜蜂汇云调用我们的放单接口
  - 我们处理充值后回调状态给蜜蜂汇云

- **客客帮**：采购商
  - 我们对接客客帮的**供货商API**
  - 客客帮调用我们的订单接口
  - 我们处理充值后回调状态给客客帮

### 2. 供货商管理
**我们作为采购商**

- 从外部供货商采购商品
- 配置外部供货商的API接口
- 我们作为买家角色

## 📊 对接流程

### 1. 接收订单流程

```
采购平台（蜜蜂汇云/客客帮）
    ↓ 调用放单API
我们的系统
    ↓ 处理订单
    ↓ 执行充值
    ↓ 完成充值
采购平台
    ↓ 回调状态
订单完成
```

### 2. 商品同步流程

```
我们的系统
    ↓ 上传商品信息
采购平台
    ↓ 审核商品
    ↓ 上架
商品可售
```
