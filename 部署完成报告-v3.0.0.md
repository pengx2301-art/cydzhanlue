## 🔧 部署步骤回顾

### 1. 上传文件
```bash
scp -i chongyida.pem api-server-v3.js ubuntu@81.70.208.147:/app/
scp -i chongyida.pem user-backend.js ubuntu@81.70.208.147:/app/
scp -i chongyida.pem zilaidan-manager.js ubuntu@81.70.208.147:/app/
scp -i chongyida.pem supplier-manager.js ubuntu@81.70.208.147:/app/
scp -i chongyida.pem mifenghuiyun-supplier.js ubuntu@81.70.208.147:/app/
scp -i chongyida.pem user-backend.html ubuntu@81.70.208.147:/app/
scp -i chongyida.pem user-backend-ui.js ubuntu@81.70.208.147:/app/
```

### 2. 安装依赖
```bash
ssh -i chongyida.pem ubuntu@81.70.208.147
cd /app
npm install
```

### 3. 重启服务
```bash
pm2 stop api-server
pm2 delete api-server
pm2 start api-server-v3.js --name api-server
pm2 save
```

### 4. 验证部署
```bash
curl http://localhost:8899/api/health
pm2 status
pm2 logs api-server --lines 30
```
