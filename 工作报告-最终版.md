## 📊 API 端点总结

### 认证 API
- `POST /api/auth/login` - 登录
- `POST /api/auth/logout` - 退出
- `GET /api/auth/me` - 获取当前用户

### 订单 API
- `GET /api/orders` - 订单列表
- `POST /api/orders` - 创建订单
- `GET /api/orders/:id` - 订单详情
- `GET /api/orders/:id/balance` - 订单用户余额 ✨
- `PUT /api/orders/:id/status` - 更新订单状态
- `POST /api/orders/:id/recharge` - 发起充值

### 用户 API
- `GET /api/members` - 用户列表
- `POST /api/members` - 创建用户
- `GET /api/members/:id` - 用户详情
- `PUT /api/members/:id` - 编辑用户
- `POST /api/members/:id/balance` - 调整余额
- `PATCH /api/members/:id/status` - 切换用户状态

### 产品 API
- `GET /api/products` - 产品列表
- `POST /api/products` - 新增产品
- `GET /api/categories` - 分类列表

### 日志 API
- `GET /api/balance-logs` - 余额变动日志
- `GET /api/operation-logs` - 操作日志
- `GET /api/balance/overview` - 余额管理统计
