## 📞 常见问题

### Q1: PM2命令不存在
```bash
# 安装PM2
sudo npm install -g pm2
```

### Q2: Node.js未安装
```bash
# 安装Node.js
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs
```

### Q3: 端口被占用
```bash
# 查找占用端口的进程
sudo lsof -i :8899

# 杀死进程
sudo kill -9 <PID>

# 重新启动
pm2 restart cyd-admin
```

### Q4: API返回500错误
```bash
# 查看详细日志
pm2 logs cyd-admin --lines 100

# 检查配置文件
cat /app/product-connection.js

# 重启服务
pm2 restart cyd-admin
```
