## 脚本特性

- **自动重试**：每30秒重试一次，最多100次
- **网络检测**：推送前先检测网络连接
- **状态检查**：确保子模块没有未提交的更改
- **彩色输出**：清晰显示推送进度和状态
- **可中断**：按 Ctrl+C 可随时停止

## 故障排查

如果推送持续失败，可能的原因：

1. **网络防火墙**：检查是否被防火墙阻止访问 github.com:443
2. **代理设置**：如果使用代理，需要在 Git 中配置
3. **SSH 密钥**：如果使用 HTTPS，可能需要输入密码
4. **仓库权限**：确认有推送权限
