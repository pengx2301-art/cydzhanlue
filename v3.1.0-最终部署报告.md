## 🗄️ 数据库结构

### 用户表 (users)
```sql
- id: 用户ID
- phone: 手机号（唯一）
- password: 密码（bcrypt加密）
- username: 用户名
- balance: 余额
- total_spent: 总消费
- status: 状态
- created_by: 创建者ID
- created_at: 创建时间
- updated_at: 更新时间
```

### 管理员表 (admins)
```sql
- id: 管理员ID
- username: 用户名（唯一）
- password: 密码（bcrypt加密）
- name: 姓名
- role: 角色（super/admin/operator）
- status: 状态
- last_login_at: 最后登录时间
- created_at: 创建时间
- updated_at: 更新时间
```

### 供货商表 (suppliers)
```sql
- id: 供货商ID
- name: 供货商名称
- type: 供货商类型
- api_url: API地址
- api_key: API密钥
- api_secret: API密钥
- config: 其他配置（JSON）
- status: 状态
- test_status: 测试状态
- test_message: 测试消息
- created_at: 创建时间
- updated_at: 更新时间
```

### 商品表 (products)
```sql
- id: 商品ID
- supplier_id: 供货商ID
- supplier_product_id: 供货商商品ID
- name: 商品名称
- category: 分类
- price: 售价
- cost_price: 成本价
- stock: 库存（-1表示无限）
- status: 状态
- config: 商品配置（JSON）
- created_at: 创建时间
- updated_at: 更新时间
```

### 订单表 (orders)
```sql
- id: 订单ID
- order_no: 订单号（唯一）
- user_id: 用户ID
- product_id: 商品ID
- supplier_id: 供货商ID
- account: 充值账号
- quantity: 数量
- total_amount: 订单总额
- cost_amount: 成本金额
- status: 订单状态
- platform_order_id: 平台订单ID
- platform_status: 平台订单状态
- error_message: 错误信息
- created_at: 创建时间
- updated_at: 更新时间
- completed_at: 完成时间
```

### 加款申请表 (topup_applications)
```sql
- id: 申请ID
- application_no: 申请单号（唯一）
- user_id: 用户ID
- amount: 申请金额
- status: 审核状态
- note: 申请说明
- admin_id: 审核管理员ID
- admin_note: 审核备注
- approved_at: 审核时间
- created_at: 创建时间
```
