## 🔧 技术实现

### 登录流程

#### 用户登录流程
1. 用户访问 `/user/login.html`
2. 输入手机号和密码
3. 点击登录按钮
4. 调用 `POST /api/user/login`
5. 服务器验证凭据
6. 返回JWT Token
7. 保存Token到localStorage
8. 跳转到 `/user-backend.html`

#### 管理员登录流程
1. 管理员访问 `/admin/login.html`
2. 输入用户名和密码
3. 点击登录按钮
4. 调用 `POST /api/admin/login`
5. 服务器验证凭据
6. 返回JWT Token
7. 保存Token到localStorage
8. 跳转到 `/admin.html`

### Token存储
- 用户Token: `localStorage.getItem('userToken')`
- 管理员Token: `localStorage.getItem('adminToken')`

### Token使用
```javascript
// 用户API调用
headers: {
  'Authorization': 'Bearer ' + localStorage.getItem('userToken')
}

// 管理员API调用
headers: {
  'Authorization': 'Bearer ' + localStorage.getItem('adminToken')
}
```
